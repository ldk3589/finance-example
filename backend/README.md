# Finance Manager

一个基于 Spring Boot + MyBatis-Plus + Vue 的个人财务管理系统。

## 技术栈
- Java 21
- Spring Boot 3
- MyBatis-Plus
- MySQL
- Vue 3
- ECharts

## 功能
- 用户账户管理
- 收支记账（事务控制）
- 分类统计报表
- 消费结构饼图展示

## 项目亮点
- 使用 @Transactional 保证记账一致性
- 统一返回 Result 结构
- 全局异常处理
- 前后端分离设计
- 日志与单元测试支持

## 启动方式
1. 创建数据库并导入表结构
2. 修改 application.yml 中数据库配置
3. 启动 Spring Boot
4. 启动前端项目
